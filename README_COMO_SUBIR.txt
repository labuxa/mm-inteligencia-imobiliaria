MM Inteligência Imobiliária - V10 final com artigos e páginas comerciais

Arquivos/pastas deste pacote:
- index.html: página principal atualizada.
- artigos/: páginas de conteúdo informativo.
- servicos/: páginas comerciais/explicativas dos serviços.
- robots.txt: instrução para buscadores.
- sitemap.xml: mapa do site para Google Search Console.
- 404.html: página de erro personalizada.
- CNAME_PARA_USAR_DEPOIS.txt: usar apenas quando o domínio estiver liberado.

Como subir no GitHub:
1. Extraia este ZIP no computador.
2. Abra o repositório mm-inteligencia-imobiliaria no GitHub.
3. Clique em Add file > Upload files.
4. Arraste estes itens para o GitHub: index.html, a pasta artigos, a pasta servicos, robots.txt, sitemap.xml e 404.html.
5. Não envie o arquivo CNAME_PARA_USAR_DEPOIS.txt como CNAME agora. Ele é apenas uma anotação para quando o domínio estiver ativo.
6. Clique em Commit changes.
7. Aguarde 1 a 3 minutos.
8. Teste: https://abuxa.github.io/mm-inteligencia-imobiliaria/?v=10

Quando o domínio estiver ativo:
- No GitHub Pages, configure o domínio www.mminteligenciaimobiliaria.com.br.
- No Registro.br, configure o DNS conforme instruções do GitHub Pages.
- Depois envie/renomeie o arquivo CNAME com uma única linha: www.mminteligenciaimobiliaria.com.br

Observação:
Os artigos e páginas não fazem promessa de resultado e mantêm caráter informativo.
